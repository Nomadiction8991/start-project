# Projeto Docker + PHP

## Descrição

Repositório criado como **minha aprendizagem** sobre como funciona o Docker com PHP (Apache) e para servir como **modelo reutilizável** em outros projetos. Este repositório é um ambiente pessoal de estudo — não um produto final.

## Como usar (rápido)

```bash
# Subir em desenvolvimento (rebuild e hot-reload)
docker compose up --build

# Em background (detached)
# docker compose up -d --build

# Acessar localmente
http://localhost:8080

# Parar e remover containers
docker compose down
```

-----

_Pronto para reutilizar como base em novos projetos PHP com Docker._
